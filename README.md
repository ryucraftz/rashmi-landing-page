This project is ready for immediate deployment on Vercel.

### Option 1: Vercel CLI (Recommended for fast deploy)
1. Install Vercel CLI (if not installed):
   ```bash
   npm i -g vercel
   ```
2. Run the deploy command in this directory:
   ```bash
   vercel
   ```
3. Follow the prompts (Keep default settings).

### Option 2: Git Integration
1. Push this folder to a GitHub/GitLab repository.
2. Go to [Vercel Dashboard](https://vercel.com/new).
3. Import your repository.
4. Vercel will automatically detect it as a static site. Click **Deploy**.

## Project Structure
- `index.html`: Main landing page.
- `style.css`: Custom premium styling.
- `script.js`: Smooth scrolling interactions.
- `vercel.json`: Configuration for clean URLs.
